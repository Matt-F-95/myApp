What is Electron?

Electron is a desktop application creator program, It reads programs written in JavaScript and then displays them in the chromium browser during the creation process. The apps created with Electron can be turned into an executable file(.exe) that will run on Windows or Mac using the computers native browser window.

The executable file can be created with an external program called Node.js which basically packages the files made with Electron.

Electron runs the main javascript file that is inputed into the package.json file, within this main.js file is where you create your BrowserWindow and can make changes to the rendering process also.

Electron is also an open-source project developed by GitHub.
